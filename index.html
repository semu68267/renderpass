렌더링 빠른 견적

선택할 때마다 실시간으로 금액이 계산됩니다. 단위는 만원임.

프로젝트 유형
공간/대지 범위에 따라 기본 난이도가 달라짐.
표현 레벨
베이식(저품질) / 프로(고품질) / 엘리트(하이엔드).
자료 제공
가로축: 도O·모O / 도X·모O / 도X·모X 기준으로 가격 매핑됨.
예상 견적
컷 1장 기준 · 부가세 별도
- 만원
선택 요약
옵션을 선택하면 요약이 표시됨.
TIP “자료 없음(도X·모X)”일수록 모델링·리서치 비용이 포함되어 단가가 상승함.
참고사항
  • 도시 야경, 복잡한 파사드, 광활한 부지 등은 추가가 발생할 수 있음.
  • 컷 수가 늘면 묶음 할인 가능. 시안·수정 라운드 정책 협의.
  • VR/파노라마, 동영상, 드론 합성 등은 별도 문의.
